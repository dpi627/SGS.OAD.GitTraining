# 📚Git 心得分享與實作

這個 repo 放置內部 Git 分享資料，包含

- 簡報原始檔，以 `markdown` 撰寫
- 簡報發佈檔，以 `pdf` 發佈
- 部分基礎操作以 `mp4` 錄製
  
## 💡如想直接檢視簡報
- 快速上手基礎版控 [Quick Start](publish/quick-srart.pdf)
- 完整檢視內訓簡報 [請點此](http://twvoadtpw100004/brian_li/SGS.OAD.GitTraining/src/branch/main/publish)

# 🏗️專案架構

```c
📁 asset        //圖檔、影像檔
📁 public       //簡報發布檔 *.pdf
📁 src          //簡報原始檔 *.md
📄 README.md    //本檔案
```
