# Git 心得分享與實作

這個 repo 放置內部 Git 分享資料，包含

- 原始檔案，以 `markdown` 撰寫
- 發佈簡報格式為 `pdf`
- 包含多份資料，完整分享約 8hr

# 專案架構

```c
📁 asset        //圖檔、影像檔
📁 public       //簡報發布檔 *.pdf
📁 src          //簡報原始檔 *.md
📄 README.md    //本檔案
```
